【魔都同人祭COMICUP04 - 端午粽子节 —— 交通指南】
时间：5月28日 10:00 - 17:00 (16:00 停止入场，若有特殊情况，将适当提前开场)
地点：上海世贸展馆3楼东厅 3800平米 超大面积升级醒目！！

地址：上海市长宁区兴义路99号（打车关键字：世贸商城、兴义路-娄山关路） 地铁二号线 娄山关路2号口出，沿娄山关路向南800米，约2条横马路后至兴义路口即是。近虹桥、东华大学。

【相关交通】
>>> 轨道线：地铁2号线（娄山关路站），或在地铁2、3、4号线中山公园站 高架下搭乘【免费传送车】
×娄山关站 2号口出站，沿着娄山关路一直向南（延安西路高架），过两条横马路，到兴义路口向右，就可以看到了。
——人肉实测步行 约 16分钟（含买了奶茶，和人聊天，同时还边走边逛吃的……）

×中山公园站（3号线、4号线站点）下站后至高架下寻找橙色标志，我们将安排小精灵直通车，将你从地铁站送到会场。XD

×上海世贸商城有延安西路2299号、兴义路99号2个入口；推荐的入口是兴义路99号，请注意下图蓝色标志和售票厅上的橙色小横幅。

* 迷路了，请找橙色袖章的亲切大姐姐大哥哥。NPC将带您到COMICUP04登陆点。

公交站③——虹桥路古北路
公交48路[石门一路至上海动物园]、827路[番禺路至桃浦路]、938路[虹桥机场至浦东八佰伴、杨家渡]；




【外地的同学们看这里】>>>

# 上海火车站：向南广场走，出站后左手转弯找地铁三号线，乘坐至中山公园站，换地铁二号线至娄山关路站，然后同上；
# 上海火车南站：出站后乘地铁三号线至中山公园站，换地铁二号线至娄山关路站，然后同上；
# 汽车南站：出站后问问亲切的警察叔叔走到火车南站坐地铁，然后参照上一条；
# 虹桥机场：搭乘925、945、911，至虹桥开发区站，下车就能看到； 或直接TAXI，报关键字“世贸商城”，估费16元。
# 浦东机场：从磁悬浮（浦东国际机场站）上车,坐1站至龙阳路站下车,同站换乘地铁二号线至娄山关路站，然后同上。




【其他交通方式】>>>
# 东华和工技大老校区的同学可以走过来，当然自行车也行；
# 有私家车的可以直接开过来，门口左侧有停车位，只要别挡了卖票口；
# 有钱人可以直接TAXI，关键字：上海世贸展馆、世贸商城、兴义路-娄山关路口；
# 有小叮当的可以使用任意门，记得买好铜锣烧；
# 有高达的同学……门口左侧就卸货停靠平台，驾驶舱高度则直达COSPLAY专区落地窗。
