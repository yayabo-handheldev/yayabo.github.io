[edit] Basic Usage
Left shoulder goes to the previous page, Right shoulder goes to the next
Start brings up the open file dialog.
Select brings up an information console for debugging
X changes the orientation of the document
Grab and pan the document with the stylus
Click the bullseye button to see a mini-document view; tap it to go to that part of the document and draw a line to zoom to fit that line.
[edit] Common Problems and Known Issues
Large files or files with many images will cause the program to run out of memory and quit.
[edit] Additional Information
If you encounter any problems with the program, please contact the author through the ands-pdf blog
