【转自巴士】
国外网站WarioWorld.com是一家专门提供任天堂硬件与及软件支持的小组，今天该小组的主页增加了一个新的登入口，而这个登入口的图标就是任天堂3DS。一些没有得到任天堂特别邀请的第三方开发商，可以通过登录这里获得相应的工具包。这次3DS技术的开放也预示着3DS游戏开发正式进入白热化，受益的将会是所有3DS的玩家！

　　该小组也曾经提供过以下平台的技术支持

　　Nintendo GameCube
　　Game Boy Advance
　　Nintendo GameCube - Game Boy Advance Link Cable
　　Nintendo 64
　　Game Boy Color
