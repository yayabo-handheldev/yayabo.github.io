==Usage== 使用方法

==Options== 选项

There are a few options available in RomeR they mostly deal with
the header. I advise using the new slider to alter the settings.
Dafault settings have worked for all tested homebrew apps but as
many users requested info on the options here it is!
RomeR中有一些选项，在处理文件头的时候尤其有用。
我建议使用滑动条来更改设置。
默认设置已经通过了所有自制软件的测试。
而且也实现了很多用户要求的选项要有描述性文字这一需求。

Fix header checksum:
This option calculates the logo checksum then writes this to
memory and calculates the second checksum for the whole header
( the first checksum result is included in the calculation)
修复文件头校验码：
这个选项会计算logo的校验码，然后写到内存，再计算整个文件头的校验码。
（第一次计算的校验码会被包含到第二次计算的过程中）
注意：这个功能没啥作用。对于烧录卡来说，文件头校验码实在是没啥意义。

Fix header length pointer:
This pointer tells the NDS where the end of the .nds file is
when enabled RomeR determines the exact size of the file and
writes the value into the header
修复文件头长度指针：
如果激活这个功能的话，这个指针会指出NDS文件的真实大小，然后写到文件头里面。
注意：这个功能也没啥用。

Fix unit code option:
The header contains a unit code this option forces it to equal
00 which is the normal value
修正UnitCode选项：
文件头包含一个叫做UnitCode的字节代码。当这个代码为00的时候，ROM被认为是正常。
这个功能会令该代码强制为00。
注意：这个功能只对官方游戏有意义，对于自制软件依然没有意义。

Force header size option:
This forces the header size variable to be 0x4000 this is the
normal value for games, many homebrews have a header size of
0x200
强制文件头大小选项：
这个选项会强制令文件头的大小为0x4000。
注意：这个功能也只对官方游戏有意义。一般的自制软件文件头大小都为0x200，但这无所谓，反正DS能够运行……

Force logo CRC:
This option this forcefully inserts the nintendo logo therefore
making the logo checksum always equal to 0xCF56.
强制logo的CRC校验
这个选项会强制修复文件头的任天堂logo数据，从而使logo的CRC校验值为0xCF56。
注意：在某些烧录卡上，尤其是早期的烧录卡，最好使用这个功能修复一下。
最近的烧录卡貌似没必要使用这个功能，不过修复了也没坏处……


==1==
Load a rom either from file menu or via the browse button
菜单或者“browse”按钮载入ROM

(Homebrew roms will be detected and the homebrew checkbox will automatically be ticked for you)
（如果是自制软件的话，会被自动检测出来。此时homebrew的复选框会被自动选中）

*NOTE* homebrews previously edited in RomeR do not require the homebrew app checkbox to be selected.
注意：如果这个ROM以前曾经被RomeR修改过的话，那么那个复选框可以不用选中

==2==
Edit the Titles and other information as desired
编辑ROM的标题文字和其他描述信息

==3==
Select an input image, it should be 32 x 32 in size and have a color pallete of 16. (Suggest using a 16 color gif image)
选择一个图片当做图标。图片要求是32*32像素，颜色最多为16色。注意要用“索引色”图片，不能用“RGB”图片（建议使用16色的单帧GIF）

==4==
Select apply changes and wait for the rom edited box to appear. May take a while on a very slow computer
点击“Apply changes”按钮，万事大吉
