To use this patch you will need to load it using a regular Nintendo DS, DS Lite or DSi with firmware 1.4x or lower.
Simply run the appropriate file attached in this thread, eg. DSL.nds if you're running the update on a Nintendo DS or DS Lite or DSi.nds for if you're running it on a Nintendo DSi 1.4x or lower.
Carefully follow the onscreen instructions and within minutes your Acekard 2i will be 1.4.1 compatible
