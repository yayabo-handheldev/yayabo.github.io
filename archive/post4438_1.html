━━━━━━━━━━━━━━━━━━━━━━

软件开发：きまぐれ
原作者地址：[url]http://takka.tfact.net/blog/[/url]

ISO Tool 是日本某程序员 きまぐれ 开发的一款破解程序，可在PSP上直接破解ISO/CSO的“EBOOT.BIN”，无需导出EBOOT.BIN，免去了PC操作的麻烦。

ISO tool主要功能：

- iso压缩为cso，可以选择压缩等级压缩阈值等。
- cso解压为iso，给小p运行时减负
- umd提取为iso/cso，给umd光驱减减负
- iso/cso直接本机破解，补丁，备份eboot，备份还原
- umd/iso/cso信息显示（这个很绝吧，就算iso文件名乱取也可以显示游戏名称咯，不过需要一个叫 umd_id.txt的文件更新来提供这些信息）。
- iso/cso文件放在任何地方都能破得到 在线更新umd_id.txt 在线升级主程序！
- 可以让你的机器在5.XX系同级系统之间自由切换，相当于软刷机。
而且由于不更改核心成分，所以它软刷出来的普米系统比一般普米系统更加稳定，可惜也由于这个原因，软刷的普4不支持中文名称。
5.00可以在5.00 M33-6，5.00 M33-6at，5.00 Prometheus3，5.00 Prometheus4之间切换。
5.03可以在5.03 GEN-C，5.03 GEN-Cat，5.03 Prometheus3，5.03 Prometheus4之间切换。
5.50可以在5.50 GEN-D2，5.50 GEN-D3，5.50 GEN-D2at，5.50 GEN-D3at，5.50 Prometheus3，5.50 Prometheus4之间切换。
补充说明：就算加了普米补丁，不加AutoCFW的非普米系统也仍然是运行不了游戏的。
5.00 M33-6at里的“at”指的就是AutoCFW。ISO TOOL的原下载包里带了那个插件。

[b]OFW版[/b]：已签名，直接在未破解的官方系统都可以运行；已增加ME631补丁。

[b]CFW版[/b]：只能在破解的PSP运行；已增加ME631补丁；带AutoCFW插件，可以运行怪物猎人3，直接覆盖的话会强制停止已开启的插件。

[b]教程：[/b]
软件安装：

　　1.下载文件，将其解压，得到“PSP”、“seplugins”两个文件夹。

　　2.使用USB线连接PSP和电脑，将“PSP”、“seplugins”这两个文件夹拷贝至PSP记忆棒的根目录下覆盖即可（覆盖不会改变文件夹内的原文件）。
　　注：不想覆盖的需要在根目录“seplugins”下的“game.txt”文本下追加一行“ms0:/seplugins/peagasus.prx 1”即可。使用PSPgo的玩家在相同位置里的“game.txt”追加“ef0:/seplugins/peagasus.prx 1”

　　3.使用ISO TOOL破解前，请先看一看你自己想要破解的ISO〈游戏〉的属性有没有选择唯读，如果有，请取消唯读，否则会破解失败。

　　该软件有英/日两种语言，如果你的PSP主机系统语言是日语，那软件的界面语言就是日语。如果你的PSP主机系统语言不是日语，那软件的界面语言则为英语。

　　补充说明：使用该软件破解好的游戏在运行时，请先关闭GAMES中“KHBBS”插件，否则会冲突，出现黑屏等不能运行的现象。
　　插件关闭方法如下：
　　PSP3000/2000V3用户：在PSP界面下按Select呼出VSH菜单，选择RECOVERY MENU（恢复菜单），进入之后，选择CPU Speed的选项，进入之后会看到你现在所有的插件，将其中的KHBBS_patch设为Disable即可。（注：开启插件操作方法同上，只需将Disable设置为Enable即可）
　　PSP1000/2000非V3用户：按住R键不松，重启PSP主机，进入恢复菜单，选择Plugins，进入之后会看到你现在所有的插件，将其中的KHBBS_patch设为Disable即可。（注：再次开启该插件操作方法同上，只需将Disable设置为Enable即可）

[b]破解方法：[/b]

　　1.安装完软件之后，打开PSP游戏目录就能看到ISO TOOL的图标了，按O键进入软件。

　　2.选择记忆棒中的你想要破解的游戏的原版ISO，将光标移动到该ISO上，按O键出现菜单。菜单分别是“打补丁、破解EBOOT、还原EBOOT、游戏升级、创建游戏升级目录、重命名、转换成CSO、插件导及删除”。

　　3.一般是从破解EBOOT开始，选择“EBOOT decrtpt”，按O键确认。此时提示你是否备份原版EBOOT文件。根据情况进行选择。接下来提示你是否进行版本欺骗，基本上选择“Yes”，但如果要使用普罗米修斯补丁，这里选择“NO”，如果你的系统是500 M33的话，这一步请选择“YES”。然后是问你是否可以开始破解，选择YES开始破解。软件就会破解游戏的EBOOT文件，出现“Finished”字样就完成EBOOT文件的破解！

　　4.破解完EBOOT文件之后，退出软件运行游戏，假如执行游戏后出现80020148的错误代码提示无法运行游戏，想不使用插件进行游戏，我们还需要继续使用ISO TOOL，使用普罗米修斯模块来继续破解ISO，因为部分游戏ISO还需要打上普罗米修斯补丁或者针对ISO推出的特殊补丁，才能正常运行，所以还需要给该ISO打上普罗米修斯模块补丁，下面我们继续来破解ISO。

　　5.破解完EBOOT之后，在ISO上按O键出现菜单，在主菜单中选择“PATCH”，然后选择“Patch Prometheus”选项

　　6.接下来会进入到选择补丁的界面，补丁一共有15种
　　　05/ = 普罗米修斯(Prometheus)补丁第五版
　　　06beta1/ = 比05版多了几个新游戏的支持，支持大多数游戏破解
　　　ACX2/ =《Ace Combat X2》用
　　　DBTV/ = 《龙珠TAG VS》用
　　　DIVA2/ = 《初音未来 女歌手计划 2》用
　　　djmax3jp/ =《DJMAX 携带版3》日版用
　　　KHBBS/ = 《王国之心 梦中诞生》用
　　　ME631/ = 集合有6.31系统重要模块的普罗米修斯(Prometheus)补丁，用〈针对《怪物猎人P3》花屏、DJMAX 韩版 闪屏、《第二物语》语音动画不全的问题〉
　　　mysis/ =《俺妹本篇》、《俺妹制作EX》用
　　　ooo/ =《添加假面骑士 极限英雄OOO》用
　　　prometheus3_v3/ =普罗米修斯(Prometheus) 3 补丁的第三次修改版，不常用。
　　　prometheus3_v4/ =普罗米修斯(Prometheus) 3 补丁的第四次修改版，不常用。
　　　prometheus4/ =普罗米修斯(Prometheus)补丁第四版，怪物猎人 3 便携版（Monster Hunter 3 Portable）专用补丁
　　　srw_z/ =《超级机器人大战Z》、《高达回忆录》用
　　　totwrm3/ =《世界传说 光明神话3》用
　　　PS：需要使用普罗米修斯补丁破解游戏有：《王国之心：梦中降生》、《噬神者》、《职业棒球之魂2010》等，而PSP《波斯王子：遗忘之沙》破解EBOOT之后，普罗米修斯补丁对于无用，需要用到插件才能运行。最新更新需要使用普罗米修斯补丁还有《金色琴弦3》（引导模式改为NP9660即可），此外《新世纪福音战士 序》打上普罗米修斯模块之后，可以在5.03GEN-C系统上正常运行。

　　7.出现“Finished”字样就表示解密完成！接下来就退出ISO TOOL，开始运行游戏了。最后提醒一下大家，破解完游戏之后，如出现黑屏现象的，请先将王国之心的破解插件关闭，否则会冲突无法运行游戏。

　　（注：必须是ISO TOOL所支持破解的游戏原版ISO才能顺利破解，如ISO TOOL目前无法破解的游戏，敬请期待新版的ISO TOOL更新）
