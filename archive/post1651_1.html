首先，介绍一下我要介绍的软件，WabbitDS，该软件是德州仪器的图形计算器TI-83 plus图形计算器的模拟程序。


TI-83 Plus图形计算器易于使用，它提供了11到18岁中学生数学及科学课程所需要的全部功能。同时它还含有高级统计与金融功能。


通过使用闪存技术，您可以在TI-83 Plus上安装应用软件（Apps）或数学与科学以外的功能模块，从而可长期使用。TI-83 Plus教师用计算器和显示屏使还您可以将这个计算器作为全班教学和演示的工具。

如欲知道更详细的内容，可以访问以下地址：
[url=http://education.ti.com/educationportal/sites/CHINA/productDetail/cn_ti83p.html]http://education.ti.com/educationportal/sites/CHINA/productDetail/cn_ti83p.html[/url]

可以说，这款软件可是大有来头！下面，我就来给大家介绍一下，看看这款软件是否如TI-83 Plus一样让人心动。

下载软件，解压缩。（废话！）将文件名为WabbitDS.nds的文件打上DLDI补丁，传到存储卡上。当然，准备工作还没有做完，由于只是个模拟程序，还需要去官方网站下载TI-83 Plus的Operate System.
[url=http://education.ti.com/educationportal/sites/US/productDetail/us_os_83plus.html]http://education.ti.com/educationportal/sites/US/productDetail/us_os_83plus.html[/url]
文件名为TI83Plus_OS.8Xu，好了，准备好机器，充足电，开始神奇的计算器之旅。

1．基本操作篇

开机后执行WabbitDS.nds，（本人是用AK+，所以一下说明以AK+为例），指定存档类型，随便多大都能正常运行。开机后，出现以下画面：









按A或者触控点击浏览到TI83Plus_OS.8Xu，









执行以后，出现读取画面，






读取后就可以了，会在下屏出现计算器按键布局。











上屏出现：“TI-83 Plus 1.19 RAM cleared”









根据软件的缺省定义，游戏机的按键被如下定义：
A = Alpha
B = 2nd
X = Mode
Y = XTON
Start = Enter
Left = Left
Right = Right
Down = Down
Up = Up

Select = Menu
L = REWIND
R = SPEED UP
R+L = Rotate layout(You must press R first) (旋转按键布局，必须先按R)
Closing Lid = Sleep and Save （合上主机上盖= 睡眠并存档）
按A出现如下画面




按B出现如下画面




按SELECT出现如下画面









程序第一次执行后，会在存储卡根目录生成TIDS.CFG文件








并在程序所在文件夹生成TI83Plus_OS.tisv文件








生成这个文件后，就不再需要原始OS了，不过TI83Plus_OS.8Xu




也就几百KB，现在大家存储卡那么大，也不在乎了，留着以备不测。

2．程序安装篇

得益于TI-83 Plus 的强大数据及图形处理功能，可以在上面安装数量可观的应用程序和游戏，应用程序都是提高办公应用效率的优秀程序，游戏都是锻炼脑子的益智游戏，当然，这些程序和游戏都是英文版的，需要有一定英文水平。如有需要，可以到以下地址下载：
[url=http://education.ti.com/educationportal/sites/US/sectionHome/download.html]http://education.ti.com/educationportal/sites/US/sectionHome/download.html[/url]

例如下载了元素周期表periodic.8xk



传到存储卡里面，进入程序，按SELECT调出系统菜单，选择File Sending





浏览到periodic.8xk，执行，经过读取后，就可以在程序里面调用了。点击按键，看到了吧！









进去后，可以用了（里面英文显示好粗糙啊！）。
上屏下部的菜单对应下屏上部的按键。
