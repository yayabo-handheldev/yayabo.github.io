使用前的准备

SvSIP是法国人Samuel开发的一款NDS用网络电话自制软件。只要你的NDS能够通过无线网络连接到Internet，那么你就能够使用这款软件来使你的NDS变成一部网络电话。要想达到这个目的，请确认你有以下条件：

----NDS/NDSL/IDS/IDSL主机一台
----可以运行SvSIP自制软件的烧录卡一张
----用于修改config文件的PC一台
----可以连接到Internet的无线网络（不一定非得出国，也就是说教育网也可以）

备齐了以上东西，那就正式进入我们的教程吧~


如何使用SvSIP

首先，你需要下载SvSIP这个软件……（众人云：废话！）

将下载下来的压缩包解压缩后，你会看见这些文件：

[attach]2524[/attach]

其中，svsip文件夹里面的文件是用于参数设置的一些文件，svsip.nds是主程序，wifi-config.nds是用于设置NDS WiFi连接的一个小软件----当然，你也可以用诸如马车等游戏来完成你的网络设置，Readme.txt是作者关于本软件的一些说明。我们需要拖到卡里面的东西是svsip文件夹和svsip.nds，当然之前要记得给svsip.nds 文件打上合适的DLDI补丁。另外，如果你需要用wifi-config.nds来设置网络的话，也把它一起拖到卡里吧。需要注意的是，svsip文件夹需要放在根目录，至于svsip.nds嘛，随你喜好咯~
做好这一切之后，先别慌着把你的储存卡从电脑上拔下，我们还需要作一些设置。
打开svsip文件夹，你会看见这些文件

[attach]2525[/attach]

我们需要修改的是config.txt文档。（网络设置我还是推荐在游戏里面搞定，毕竟那个界面看起来要简单一些）双击打开文档，你将会看见一堆英文。下面我以快门shutter为例，来讲解如何设置SvSIP。当然，这个软件是一个通用型SIP电话软件。也就是说，只要是SIP网络电话，都可以用这个软件来登录，不管是快门Shutter还是ToCall什么的。（Skype应该也是可以的，不过我没有账号就没有测试了。如果哪位有Skype的帐号，希望可以发一个测试结果上来）用快门Shutter来做例子仅仅是因为它在教育网有服务器，绝无广告嫌疑……（众人云：此地无淫XX两……其实我也是为了做这个才知道快门的……Orz……偶素无辜的）下面我们就一项一项来看：

[quote]# This is a comment in the config file.
# Information on [url]http://svsip.free.fr[/url][/quote]
这两行……基本介绍，无需修改……呵呵~
[quote]# Set domain sip provider (example: freephonie.net)
--domain XXXX[/quote]
这行是设定你要登录的服务器。把XXXX替换成网络电话运营商的服务器地址即可。比如Shutter在教育网的服务器是sip.etone.edu.cn，把这个地址替换掉XXXX就可以了。具体每个网络电话运营商的服务器地址可以到运营商的官方网站去查看，在PC端网络电话软件的设置选项里面应该也有。注意一定要选择与自己网络匹配的服务器，比如你是电信就选择电信服务器，是北方教育网就填教育网或者网通的服务器地址。这个会直接影响你能否正常使用网络电话。当然，如果服务器本身就是多线的，就当我什么都没说……Orz……最后，端口号不需要填……
[quote]# Set authentication username
--username XXXXX[/quote]
设置你的用户名。就是你登录网络电话的用户名。一般说来应该是一串数字。替换掉XXXX即可。
[quote]# Set authentication password
--password XXXXX[/quote]
设置密码。这个不用我多说吧？
[quote]# Optional registration interval (default 55) (for freephonie set 1800)
--reg-timeout 1800[/quote]
设置登陆间隔时间。把1800改成55吧。其实1800也没什么的……修改范围0-3600。
[quote]# Optional microphone volume (default 1.0)
--mic-level 3.0[/quote]
NDS麦克风的音量。原来是3.0。可以说是相当灵敏，个人认为不需要改了。当然，如果你所处的环境比较嘈杂，可以适当改小一些。修改范围0-10。
[quote]# Optional speaker volume (default 1.0)
#--speaker-level 1.0[/quote]
NDS扬声器的音量。默认是1.0。如果你所在的环境比较嘈杂，可以适当改大一些。修改范围0-10。

以下这些是高级设置，不改的话不影响正常使用的。（其实是我不知道它们是干什么用的…… 汗……）
[quote]# Optional for expert: set realm
#--realm[/quote]
设置域。不明白有什么用。
[quote]#optional:
log to filename
#--log-file directory/filename
# optional: set log max level to N (0(none) to 6(trace))
#--log-level[/quote]
这两项貌似是设置登陆信息之类的。我不太明白是干什么用的。如果谁明白的话还望不吝赐教~

OK，到此为止，我们的设置就算是完成了。如果你的NDS网络设置已经完成，那么运行软件之后，NDS会自动寻找无线网络并登陆到你所设定的服务器，整个过程非常快，不到3秒钟。如果你无法登陆的话，可能是你的网络设置有问题，或者服务器与你当前的网络不匹配造成的。如何设置网络不在本文的讨论范围之内，否则不知道又要写多少了……

进入软件之后，上屏黑乎乎的一片……（众人云：好难看。好可怕。好恐怖。好怪蜀黍。汗……）下屏显示如下：

[attach]2526[/attach]

左边的数字键盘就不用说了吧？呵呵~右上角的图标是拨出电话，下面那个是挂断电话，右下角的图标是删除一个数字。目前的界面还很简陋，上屏只是显示数字及一些简单的信息而已。如果你只是NDS to PC的通话，你只需要拨打对方的网络电话号码即可（当然是同一运营商的，而且对方需要在线）。如果你要NDS to Phone，则需要看看你使用的网络电话运营商的说明，一般需要在号码前面加0或者末尾加#什么的。具体资费根据运营商的不同而不同，不过肯定比传统电话低啦~如果你帐号的余额不足是不能NDS to Phone的~这个可不是NDS的问题哦~呵呵~
大家最关心的通话质量嘛，怎么说呢，个人认为在PC上使用网络电话是什么效果在NDS上就是什么效果，完全不用担心。延迟肯定是有的，那取决于你的网络条件、运营商服务器等等多方面因素，不过我认为没什么问题。总之，各位放心地用吧。

软件也有一些按键操作，有一些很方便的功能，具体如下：
A：呼叫或接听
B：挂断
[start]+[select]：重启软件
[A]++[X]+[Y]：关机

下面我要说的是关于软件如何更换皮肤……

经过研究之后发现，其实下屏显示的图片就是svsip/numpad.png这个文件。也就是说，只要你用相同格式相同分辨率的图片替换掉原来的图片，就可以实现下屏的换肤功能。不过要注意的是，按键位置不能变哦，因为触摸屏每个区域所对应的功能可不是换个图片就能解决的……汗……不过那黑乎乎的可怕的恐怖的怪蜀黍的上屏，还没发现怎么改……
作者一共提供了12张皮肤图片，我已经将它们打包好了~大家直接将图片名改成numpad.png替换掉原文件即可。


一点小抱怨

恩……要说需要改进的地方嘛，其实还是很多的，毕竟这个软件还处于起步阶段嘛~首先是上屏还需要改进，黑乎乎的确实不好看……其次是希望作者可以加入联系人列表功能，可以添加/删除联系人，这样拨号起来也方便一些。作者在自己的网站上提到他已经在考虑这个问题了，也许下一个版本就会实现了哦~


最后我想说一说写本文的初衷。目前很多大学校园里都有了免费的无线网络，但是限于教育网的局限性（当然，有些学校的网络还是可以出国的），我们手中的NDS是无法WiFi的。不过这并不代表我们NDS的无线网卡就此歇业，许多用到无线网络的NDS软件其实都是可以在教育网内使用的，比如这个电话，比如MSN，比如浏览器。可惜许多自制软件都是外国人开发的，很多玩家不愿意去研究怎么使用，写这个的目的就是为了让大家手中的NDS能够有更多的用途，毕竟一台异质主机应该是有很多可能的，这也是标题“无限你的NDS”所要表达的意思。初步打算是介绍一些NDS的常用软件，不过更新时间不定，毕竟现实中还是有很多事情要做的~最后，祝大家打电话愉快~呵呵~
