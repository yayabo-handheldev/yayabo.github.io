[b]指证[/b]：[font=仿宋_GB2312]七楼老汉是凶手[/font]
[b]作案过程[/b]：
[font=仿宋_GB2312]1、凶手从一开始并不在七楼，而是(9:30am以后)从楼下上四楼找死者。
2、借不到钱，并且被恶意拒绝，怀有杀意，在死者饮用的水里下毒，然后(10:30am的时候)凶手当着死者面离开。
3、上楼看见六楼门口开着，为了保险起见，便打一个无声电话引开注意力，然后(10:35am)从楼上下来假装投诉以制造不在场证据。[/font]
[b]反驳指证的证据[/b]：(应该还有漏的= =b)
01、公寓 -> 统一了上下楼层大致的外形设计
02、死者的惨叫声 -> 设定死亡时间
03、死者的白领的身份 -> 为作案动机提供条件
04、死者身穿西装 -> 死者原本准备外出/排除自杀可能
05、正门是反锁的 -> 死者怕凶手突然回头借钱
06、死者中毒身亡 -> 行凶可以不在室内
07、大音量的电视节目声 -> 掩盖了凶案现场的声音
08、无声电话 -> 用作吸引六楼的人的注意力
09、借钱 -> 作案动机
10、大麻 -> 为作案动机提供条件
11、"老汉"所形容的形象 -> 为凶手的行动能力作限制
12、老汉下6L投诉太嘈 -> 制作不在场证据
13、师奶与夫人在1L聊天 -> 对凶手的出入进行限制的假象
14、胖宅的踩屎事件 -> 对配角的行动进行限制
15、夫人身上的钥匙 -> 制造只有夫人能出入凶案现场的假象
16、6L开敞着的门口 -> 对凶手的出入进行限制的假象
17、6L搞聚会 -> 对配角的行动进行限制
18、众人现场留下的痕迹 -> 扰乱指纹或者其它的搜集
19、其余嫌疑者的情报 -> 扰乱推理用
