把NO.INI打开可以看到有许多等式，其中有一个等式是 SAV/SNA File Format == Compressed，把Compressed改为Raw（注意大小写），即 SAV/SNA File Format == Raw，
然后把要转化的sav存档复制到No 2.6a文件包下的BATTERY里（当然游戏名和sav名字要一样），打开nds游戏，模拟器就会自动读取存档。运行NO.EXE读取存档再重新记录一次就大功告成了。
