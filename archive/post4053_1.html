HOW TO INSTALL/RUNPut your ISO files in the ISO folder of your memory stick. Unzip the archive in your PSP/GAME folder. Run as any normal homebrew. For TN Hen users, you of course need to run the Hen before that.
If you want to use the M33 driver, you need a copy of 5.00_isofs.prx which is not included in this package. You can get this file from the official firmware, by using PSARDumper

安装：把5.00版本固件的isofs.prx解密后改名为isofs_500放在Prometheus_Iso_loader目录中后就可以用m33 driver了。

普罗米修斯iso loader是为6.XXHEN准备的ISO Loader。

能让6.XXHEN上运行ISO，在6.20HEN下也支持从usbhostfs上读取游戏，如同USB ISO Loader一样。

使用方法：
在PSP记忆棒根目录新建ISO目录，运行Prometheus ISO Loader，用上下键选择ISO，看提示按确认键进入游戏。
按SELECT键切换ISO 模块np9660/M33，按START键切换ISO目录（从ms0还是从host0）。

已知问题：目前有部分游戏无法运行，如一些要重启PSP的游戏，如原版假面骑士，和一些破解版汉化版游戏。
