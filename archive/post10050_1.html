原文内容由 泛游系 提供

　　第八届中国国际数码互动娱乐展览会(Chinajoy)于7月28日在上海新国际会展中心召开了。作为参展的唯一一家日本游戏公司ARC SYSTEM WORKS也派代表参加了这次展会。借助ARC SYSTEM WORKS公司内出色的日文翻译人员，我们进行了简短的寒暄，还试玩了在会场展出的BlazBlue(国内译名苍翼默示录，以下简称BB)最新版《BlazBlue: Continuum Shift》，另外，泛游系记者在会场中有幸采访到了该公司开发部长田口和宪先生，提出了一些国内玩家比较关心的问题，以及该公司对于任天堂新掌机3DS的开发计划，在这里和大家分享：

[attach]10751[/attach]

　　BlazBlue游戏画面

　　记者：ARC SYSTEM WORKS去年就开始参加中国的Chinajoy游戏展，为什么选择参加这种网络游戏为主的展会?在中国游戏市场的目标是什么?

　　田口和宪：我们参加Chinajoy的目的是在中国拓展业务，推广BlazBlue系列游戏。借助Chinajoy扩大该系列在亚洲地区的知名度。

　　记者：众所周知，中国盗版猖獗，贵公司为何仍然选择在国内发展?

　　田口和宪：我们也深知中国的盗版问题，不过更知道中国有大量支持BlazBlue的铁杆玩家。所以希望通过Chinajoy大力宣传这款作品，那些喜欢游戏的玩家自然会购买正版。

　　记者：BlazBlue系列今后的发展计划是?

　　田口和宪：目前我们已经在多个平台推出了BlazBlue系列，游戏其中包括PS3、Xbox360和PC，也包括DSiware这样的平台。从今年开始，ARC SYSTEM WORKS会在日本推出面向该地区手机用户的游戏版本，虽然BB是一款格斗游戏，不过这次的手机版更倾向于社交娱乐。另外，我们还将推出iPhone使用的程序，玩家可以通过这个小程序获得有关BB的最新情报、看四格漫画、游戏预告片以及收听网络电台等等。

　　记者：热血系列是国内玩家非常喜欢的系列游戏，贵公司在今年推出了多款该系列游戏，今后的发展计划如何?

　　田口和宪：目前热血系列将作为NDS平台独占的游戏系列，还没有发售其他平台的计划。

　　记者：对于任天堂的新掌机3DS，请问您的看法，以及贵公司的开发计划?

　　田口和宪：ARC SYSTEM WORKS不但看好3DS的后续发展，并且也一定会支持3DS!我觉得3DS是一款性能很强的掌机，相比之前的所有掌机，3DS可以做到更多之前不能做到的东西。对于ARC SYSTEM WORKS来讲，3DS是非常有开发价值的平台。

　　记者：那么，BlazBlue系列是否会推出3DS版呢?

　　田口和宪：从技术角度来讲，是可以的，完全没有任何障碍。不过目前还没有确切的开发计划。

　　记者：最后，请您对国内的玩家说几句话。

　　田口和宪：无论你身在何处，都希望各位能支持ARC SYSTEM WORKS，支持正版游戏。我们也会以更好的游戏回报各位玩家。谢谢。
