大约像是这样：
function crossback(label)
if KeyDown(KEY_CROSS) == 1 then
goto(label)
return
elseif KeyDown(KEY_CIRCLE) == 1 then
return
end
loop()
end

将这一段函数加入到am_init.lua里面（建议在最上面）

具体的CG模式的浏览模式可以这么写（仅供参考 可以换成自己的写法 红字的保留）：

fg(32,"CG1.png",1,100);fgxy(32,0,0);fgon(32);
[color=Red]sysMode(EVENT_MODE);[/color]
[color=Red]crossback(*back);[/color]

这个*back可以换成乃自己的返回标签 不用再添加等待按键（函数已经代替了等待按键）
