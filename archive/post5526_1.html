[size=7][color=Lime]【游戏安装教程 初级篇】[/color][/size]
初级篇内容转自澄空学园

方便第一次接触到GALGAME的玩家,所以某写得尽量详细点

1.解压缩
您下载到的游戏有可能是压缩包或者是镜像文件
→压缩包
压缩包是指文件扩展名为rar,zip,7z等的文件,不包括iso
若压缩包内是游戏文件,请将起解压缩至单一文件夹下,运行游戏
若压缩包内为镜像文件,请看下一步

2.加载镜像
→镜像文件
镜像文件是指扩展名为iso,img,bin等的文件
这类文件请先用daemon等虚拟光驱程序加载

3.游戏安装
→情况一
双击虚拟光驱图标或者右键打开寻找setup(或类似)文件安装游戏...一路下一步...安装成功
→情况二
以上操作均无反应,那么您可能需要AppLocale
※或者您可以使用真神大作的soraapp,使用方法和下载地址那帖中都有,用这个可以避免某些使用app的副作用
使用时先用app加载虚拟光驱下的setup(或类似)文件,然后选择日本语安装
→情况三
如果用上面的方法还未解决问题,那么您只好通过转区来解决了
方法:在控制面板内寻找区域和语言选项(依照系统版本不同名称可能略有不同)
然后点"高级",将非unicode程序的语言改为日语
重起,再次安装您的游戏...这种情况请不要使用app

如果上面三种方法对您来说都不行...那就是您的RP有问题了...请PM某解决
有的时候安装程序会提示您桌面图标建立失败或者开始菜单快捷方式建立失败,这是由于文件路径导 致的,不会影响您的游戏正常运行
一般情况下建议您不要在安装路径中出现中文或者日文,纯英文路径最好,否则可能会导致一些问题

4.游戏运行
→情况一
双击游戏程序,正常运行,没有出现乱码,没音乐等等问题,那么恭喜你可以正常游戏了
→情况二
双击游戏程序无反应,或者报错,总之无法运行游戏
→方法一
用app选择日本语加载游戏程序运行(不是setup),顺便选上建立快捷方式,方便您以后使用
※或者您可以使用真神大作的soraapp,使用方法和下载地址那帖中都有,用这个可以避免某些使用app的副作用
许多游戏属于这种情况
→方法二
如果用app加载还是无法正常游戏...那么非常遗憾...您大概需要转区运行游戏了,方法见前
这种方法非常烦琐,但是有些游戏必须这样...例如《車輪の国、向日葵の少女》,还有light的部分游戏
→情况三
有些游戏默认安装就会出现日文/乱码路径...导致游戏时有些字出现左边多一点,右边少一点...等等情况
这时候你需要把游戏目录中的日文/乱码文件夹改为英文
例:C:Program Filesあいうgame →C:Program Filesgamegame
然后再把注册表中的相应键值更改一下
有些游戏需要这样,如《雪月華》

如果还是不能运行...请PM某解决
5.特殊情况
→游戏中无法存档
可能是游戏主目录下缺少save文件夹...有些非镜像安装/解压的游戏可能会出现此情况,重新建立一个"save"文件夹即可

【安装篇】
虚拟光驱自然是少不了~！但还是N人在问，虚拟光驱是什么- -|||
我个人极力推荐[color=Green]DAEMON[/color]，360软件管家下载安装够傻瓜的了

使用方法，以汉化版为例
启动软件后，点击其在任务栏的图标，选择“驱动器X-X（默认是没有媒体）”
然后导入下载回来的游戏的镜象文件。如果某些游戏没有扩展名（比如PIA3）或者扩展名不支持什么的也不要紧，将“文件类型”选成“全部文件”就能看到了。一般都能正常导入。nrg格式需要DAEMON3.41以上版本才支持。

然后就可以进行安装了。

如果安装途**现DISC之类字样停止了（往往一大串乱码），很可能就是叫你换第2张盘继续安装，于是以上面说的方法，导入第2张盘。继续安装

注意，在XP系统中，必须在[color=Cyan]日本语环境[/color]下安装。具体操作是：控制面板中选择“区域与语言环境”，然后在“高级”选项卡中将“中文”改成“日文”。确定后重新启动就可以了。（安装后就可以换回中文环境，不影响游戏，极个别例外）。也可以用[color=Blue]Microsoft AppLocale[/color]来进行实时语言转换，不需要重新启动。或者使用[color=MediumTurquoise]NTLEA[/color]进行安装。但也有个别游戏必须要日文版系统才能运行的。但毕竟这种游戏不多。对于一个日文游戏铁杆玩家，JWIN+CWIN是必不可少的（日文WINDOWS+中文WINDOWS）

【运行篇】
不少人在安装好，挂载APP或者NT准备玩的时候，弹出这样或那样的对话框。总结几点：
1.必须放入游戏盘。这个简单，用DAEMON虚拟一下游戏盘（一般是最后一张，也有例外）。就可以运行了。当然很多时候在游戏发售后不久就能找到免DVD补丁，一般可以在ｗｗｗ．ｈｇａｍｅ．ｃｏｍ找到，基本可以解决镜像加载问题
2.一些老游戏的系统兼容问题。如果是XP系统，可以在安装目录下找到游戏运行的主程序，在“属性”中把兼容性设置为98
3.游戏被加密，需要破解文件。或许在盘、镜象里有。在网上搜索下载也可以。

【常识篇】
1。发布者当然不会让别人下了又玩不到的吧~？！那么他们往往会在发布帖或在游戏安装文件附近加入一些文本之类的文件，自己去查看就知道该如何处理了。比如IZUMO就是这样。
2。补丁，往往就是用于破解或者升级的文件。如果没有特别指示，就是把他们覆盖掉游戏安装目录里的同名文件就OK了。比如公主骑士的补丁就是如此
3。可能还有些要安装DirectX 9才能正常运行的游戏。

【关于日文游戏无法安装问题解决方案】
最近下了几个18X游戏 有几个安装不了! 主要是ALICE的游戏
其实原因很简单在XP下 文件名与安装配置文件名称不同所以安装不了
具体解决方案有两种:
1: 将光盘镜像文件COPY到硬盘 先把安装目录的最后一个也就是游戏名称复制一下然后把文件名称都复制上去就可以了 比如说 魔女的赎罪 里面有7个文件都是魔女的赎罪.＊把他们都该成　杺彈偺孳嵾．＊　其余部分不要动只改这些就可以了　然后执行安装就ＯＫ了
2：用光盘工具直接在镜像里面改　有很多免费软件用ＢＡＩＤＵ搜索一下就能找到

【补充】
在中文环境下虚拟镜像时有时会产生问题，常见问题有两种：
1. 解压后的路径名、文件名中含有中文系统下的非法字符。比如サマー?ラディッシュ?バケーション中的“?”号，以及冴子さんといっしょ?中的“?”，这些字符虽然Winrar解压没有问题，但是用Daemon/酒精虚拟都会出问题。或者说文件找不到，或者虚拟起来光驱内一片空白。（当把这样的文件名拷贝到记事本里就会看到某个符号变成了问号，这就说明是个非法字符）
解决方法：Rename有问题的目录名或文件名，改成“.”就没问题了。

2. 解压后的文件名中带有日文名，而且是cue/bin格式。那样很可能在.cue中第一行指向的文件名是日文编码格式的，而硬盘上的文件名是中文编码格式的。这样虚拟这个.cue时就会报错说找不到镜像文件。
解决方法：用文本编辑器修改.cue中第一行对应的文件名。
举例来说，打开cue文件后第一行的格式是这样的：
FILE "儔儉僱.bin" BINARY
而硬盘上的文件实际是ラムネ.bin，那么直接修改成
FILE "ラムネ.bin" BINARY
就可以了。如果是在日文环境下就没有这个问题，已经改过的话就需要改回来。

3. 不要将.iso文件类型关联到WinRAR上！有些人双击.iso就用winrar解到硬盘，这样安装有可能出问题。
