I had some time to spare lately and I decided I'd update DSHobro.
I've received many requests to include manual configuration again.

So here is the change log for this version:
* Multi-line text support. To insert a line break, just type /n in DSHobro.
* Upgraded DSWifi library to the latest version: connecting to Wifi AP now works a lot better. Transfer rates apparently haven't improved though.
* Manual configuration is now possible. You may enter your IP & port manually -- for those who had problems with the automatic configuration.
* Improved graphics slightly and corrected a few bugs.

As always, please make sure to download the tool from THIS LOCATION ONLY. The ZIP file contains a .Exe (necessary server application) and ill-intentionned people could potentially replace it with a virus. So please, only download it from GBAtemp.net.[/quote]

使用方法：
[quote]第1步:运行" server.exe "程序. 微调的设定 然后点击"开始" ! 按钮.
这个界面是用来设置的 有打开的端口,及最大用户,默认即可,新增加了一些设置

第2步:运行把主程序dshobro.nds考入烧录卡 一般的卡用后缀为.nds的就可以了
运行dshobro.nds "启动程序,之后就可以网上冲浪了!
