　　刚刚播出的第14集的「K-ON!!」, 换了新OP及ED, 给人耳目一新的感觉, 除了带给我们惊喜, 同时也预示著「K-ON!!」也会继续在今夏的7月番里大放异彩。这相信使不少喜欢轻音五子的朋友们异常兴奋。自从「K-ON!」播出以来, 便屡创佳绩, OP及ED在音乐榜上独占熬头, BD及DVD大卖, 周边越出越多, 但仍大有市场。今次「K-ON!」又创佳绩了。

　　在6月30手发售的「K-ON! Live Event ~ Let's Go! ~」演唱会BD仅推出一周便又创了两项记录。根据著名统计网站oricon, 「K-ON! Live Event ~ Let's Go! ~」演唱会BD的首周便卖出2.2万, 立刻创了音乐类BD首週初动的记录, 把之前的记录保持者安室奈美惠于09年9月发售的「namie amuro BEST FICTION TOUR 2008-2009」(首週销量1.4万), 及水树奈奈于09年12月发售的「NANA MIZUKI LIVE DIAMONDxFEVER」(首週销量1.3万)压下。记录不单如此, 「K-ON! Live Event ~ Let's Go! ~」BD更只月了短短一週便打破了音乐类BD累积销售记录。前记录保持者同样是安室奈美惠的2.1万及水树奈奈的1.7万。

　　「K-ON! Live Event ~ Let's Go! ~」收录了声优们的主题曲及「K-ON!」动画的主题曲现场演绎版本, 仅仅用了一周便打破了安室奈美惠及水树奈奈超过半年的销售记录, 战绩真是令人惊嘆不已。同时, 「K-ON! Live Event ~ Let's Go! ~」的DVD也顺利荣登DVD总合排名第4位, 可喜可贺。
