摘自：pspchina

这是由johnmph开发的Sony PSP Application appwizard for Visual C++ 6.0.对于那些用惯了微软开发工具的人来说是个很有用处的东西.所以特此对其使用方法做了一个简单的翻译.希望能给大家提供一个帮助.但请注意,这个版本只支持VC++ 6.0,对应VC.NET的版本,作者会在不久后放出.

准备:
-----
1.TommyKat的PSPSDK 1.0+BetaWin32平台安装程序([url]http://psp.jim.sh/setup.exe[/url])
2.索尼PSP应用程序开发向导 VC++ 6.0版([url]http://www.chez.com/mph/Template.zip[/url]),源代码([url]http://www.chez.com/mph/source.zip[/url])
3.Elf2pbpm.exe([url]http://www.chez.com/mph/Elf2pbpm.zip[/url])
4.用来测试的例子工程([url]http://www.chez.com/mph/Sample.zip[/url])
5.最后当然是必不可少的Visual C++ 6.0

安装:
-----
1.把PSPSDK安装到c:pspsdk.
2.复制Elf2pbpm.zip中的elf2pbpm.exe到c:pspsdkin
3.双击执行c:pspsdkstart.bat.
4.解压Template.zip里的[color=redSony PSP Application.awx和Sony PSP Application.hlp到Microsoft Visual Studio安装目录下的CommonMSDev98Template中

使用:
-----
1.运行Visual C++ 6.0
2.文件->新建->Sony PSP Application
3.按照向导的提示,一步一步建立你的工程
4.编辑/修改代码
5.按F7建立工程

- 你可以使用上面提到的那个用来测试的例子

提示:
-----
仅适用于visual studio 6.0,对应visual studio .net的版本不久就会放出.
