由于目前的主流S1端烧录卡都不支持玩GBA游戏，所以想玩GBA游戏的玩家除了可以购买S2端DS烧录卡以外，还可以选择配合S1端实现GBA游戏，震动以及扩展内存的扩展卡，购买这样的扩展卡，除了省去升级的烦恼以外，还可以实现和S1端烧录卡的联动，以及实现DS的某些特殊功能，比如：联动、上网浏览器内存等。
　　目前，比较流行的有EZ推出的三合一工具、EZ4L、EWIN扩展卡二合一以外，还有DSTT以及M3豪华版中推出的扩展卡，一般玩家可以根据自己的烧录需要合理选择。当然，如果你是早就拥有了S1端烧录卡，只要想添置一块S2扩展卡的话，那么EZ的三合一扩展卡是你比较合适的选择。
　　EZ官方本身只提供EZ5烧录卡和EZ三合一工具的联动。但是，日本DS自制软件作者Rudolph开发出一款工具GBA ExpLoader，使用本程序，你可以通过s2端的扩展卡来运行s1烧录卡中tf存储卡上的gba游戏。如果你的s1接口的烧录卡支持软复位，你也可以将其当作震动卡及内存扩展卡来使用。
　　
　　三合一扩展卡的规格 1.烧录模式 2.内存模式 3 sram存档区
　　1. 256Mbit(32MByte) NORfalsh page size is 1Mbits(128KB)
　　2. 128Mbit(16MByte) PSRAM page size is 1Mbits(128KB)
　　3. 4Mbit(512KByte) SRAM page size is 64Kbits(8KB)
　　
　　ez4 lite-delux的规格
　　1. 384Mbit(48MByte) NORfalsh page size is 1Mbits(128KB)
　　2. 256Mbit(32MByte) PSRAM page size is 1Mbits(128KB)
　　3. 8Mbit(1MByte) SRAM page size is 64Kbits(8KB)
　　
　　ewin扩展卡的规格 1.sdram内存 2.sram存档
　　1. 256Mbit(32MByte) SDRAM page size is 16Mbits(2MB)
　　2. 2Mbit(256KByte) SRAM page size is 256Kbits(64KB) OR
　　2. 1Mbit(128KByte) SRAM page size is 256Kbits(64KB)
　　＜DSTT GBA&EXPLORER 2IN1＞
　　DSTT扩展卡的规格 1.sdram内存 2.sram存档
　　1. 256Mbit(32MByte) SDRAM
　　2. 1Mbit(128KByte) SRAMpage size is 256Kbits(64KB)
　　＜GBA Expansion Pack＞
　　M3 gba扩展卡的规格 1.sdram内存 2.sram存档
　　1. 256Mbit(32MByte) SDRAM
　　2. 8Mbit(1MByte) SRAMpage size is 256Kbits(64KB)
目前，这款软件的最新版本是0.57（文章老了，最新版是[url=http://www.yayabo.cn/thread-2526-1-1.html]0.58b[/url]），以下是DSTT烧录卡使用该软件和三合一扩展卡联动的教程。
　　1.首先对GBA_ExpLoader.NDS打DLDI补丁，（如果你的烧录卡支持自动DLDI补丁，那么你就不必打了），比如DSTT就不必打补丁。
　　2.你可以手动在TF卡根目录下建立"GBA_SAVE" 和 "GBA_SIGN"文件夹，或者你可以在DS上运行主程序，会自动生成2个文件夹。

[img=272,204]http://pic1.duowan.com/nds/0809/85593290602/85593396799.jpg[/img]


　　3.将gba文件（clean rom）拷贝到你的TF卡中。
　　4.将SAV存档文件拷到 "GBA_SAVE"文件夹下（如果没有SAV存档文件，程序会自动建立新存档文件）
　　*如果你想使用gba游戏时外框背景填充的话，将 "gbaframe.bmp 图片(24位 或 8位)"放在"GBA_SIGN"文件夹下。
　　关于GBA ExpLoader.ini文件的修改：
　　修改这个文件可以改动设置。用#或者！号加在一行的最前面可以添加一条注释行，你需要确定你将此文件放在了存储卡的根目录下
　　*SaveDir 可以改动存储SAV文件的目录（默认是/GBA_SAVE）
　　*SignDir 可以改动SGN文件以及header.dat（用处见下文）文件的存储目录（默认是/GBA_SIGN）
　　*No_MultiSave 禁用多存档功能。将其注释之后便可开启这功能。
　　5.在烧录卡界面运行 "GBA_ExpLoader.NDS"

[img=272,204]http://pic5.duowan.com/nds/0809/85593290602/85593396800.jpg[/img]


　　你可以看到如下界面：

[img=272,204]http://pic1.duowan.com/nds/0809/85593290602/85593396801.jpg[/img]


　　*在 "EZ三合一扩展卡"中, 通过L或者R键来改变模式(PSRAM模式, NOR模式或者扩展模式

[img=272,204]http://pic1.duowan.com/nds/0809/85593290602/85593396802.jpg[/img]


[align=center]　　PSRAM/SDRAM 模式[b]按键说明：[/b][/align]
　　A：将gba文件写入psram内存并打入sram补丁
　　（从sav文件读取sram数据之后，然后从psram内存上运行gba游戏）
　　B：备份sram数据到sav文件中
　　X：备份所有的sram数据到sram.bin文件中
　　Y：恢复sram.bin数据到sram中
　　R：切换到其他模式
　　在三合一扩展卡上，你可以在psram模式下运行最大到16000KB（不到16MB）的游戏，而且如果在支持软复位的情况下按SELECT键可以返回你烧录卡菜单。你会一直在内存中运行gba游戏，直到DS关机或者拔出你的扩展卡。
　　当你想交换sav文件的时候，先按B键来备份sav数据（或者运行其他的gba游戏）然后就可以交换文件了。
　　sav文件将会在每次你运行gba游戏之前被写入sram，即使你每次运行是同一个文件。注意如果你的扩展卡只有128k的sram，你就不能玩存档格式为1Mb flash的游戏。

[img=272,204]http://pic3.duowan.com/nds/0809/85593290602/85593396803.jpg[/img]


　　在上屏的菜单中，将广标移动到GBA游戏上，按A键将游戏读取进内存，等待几秒之后，就可以进入GBA游戏画面，

[img=272,204]http://pic4.duowan.com/nds/0809/85593290602/85593396804.jpg[/img]


　　如果你在psram/sdram模式下运行gba游戏，下次再运行时存档文件将会自动存储，你可以选择存在哪里或者不存档，如下图：

[img=272,204]http://pic1.duowan.com/nds/0809/85593290602/85593396805.jpg[/img]



[img=272,204]http://pic1.duowan.com/nds/0809/85593290602/85593396806.jpg[/img]


[align=center]　　nor模式（仅在扩展卡是EZ三合一卡上开启）[/align]
[align=center][b]按键说明：[/b][/align]
　　A：拷贝游戏
　　（备份sram数据到sav文件中之后，将gba文件写入nor内存并打入sram补丁，然后从sav文件读取sram数据）
　　B：备份sram数据到sav文件中
　　X：运行烧录在nor内存中的gba游戏
　　Y：从sav文件读取sram数据
　　在该模式中，中间的方框显示目前Nor内存中已经存在的游戏，如果你要玩该游戏的话，不需要再次烧录，直接按X键进入游戏，如下图。此时，按A键读取存档，或者是不读取存档，全新游戏。

[img=272,204]http://pic2.duowan.com/nds/0809/85593290602/85593396807.jpg[/img]


　　如果要重新烧录游戏的话，可以按A键。这时候，如果你是第一次运行该游戏的话，就会先对该游戏进行分析。然后先进行Nor内存的擦除，如下图：

[img=272,204]http://pic2.duowan.com/nds/0809/85593290602/85593396808.jpg[/img]


　　之后，进入游戏的烧录过程，如下图：

[img=272,204]http://pic3.duowan.com/nds/0809/85593290602/85593396809.jpg[/img]


　　当然，烧录完毕游戏之后，你可以像运行普通gba卡带一样在ds的菜单中运行nor内存中的游戏。

[img=272,204]http://pic4.duowan.com/nds/0809/85593290602/85593396810.jpg[/img]



[img=272,204]http://pic3.duowan.com/nds/0809/85593290602/85593396811.jpg[/img]


[align=center]　　扩展模式（仅在三合一卡中开启）[/align]
　　该菜单中可以改动震动等级（低、中、高），打ds浏览器内存补丁以及软复位[b]关于软复位：[/b]
　　支持软复位的s1烧录卡如下：
　　1. 本身支持的烧录卡有：R4/M3Simply、DSLink
　　2. 使用系统设置来支持软复位的有：SCDS(ONE)、N-Card（请拷贝udisk.nds到根目录下）、Acekard R.P.G（请拷贝akmenu4.nds到根目录下）、X9SD、DSTT(TTDS)
　　*如果gba游戏的存档类型不是sram，那么本软件会自动对gba游戏进打sram补丁，该软件支持的存档类型如下：
　　EEPROM_V111, EEPROM_V120, EEPROM_V121, EEPROM_V122, EEPROM_V124, EEPROM_V125, EEPROM_V126
　　FLASH_V120, FLASH_V121, FLASH_V123, FLASH_V124, FLASH_V125, FLASH_V126
　　FLASH512_V130, FLASH512_V131, FLASH512_V133
　　FLASH1M_V102, FLASH1M_V103
　　Special save type of Famicom Mini(Classic NES)EEPROM_V122, EEPROM_V124 and undefined type
　　[b]豆知识：[/b]
　　*存档文件被存在 "GBA_SAVE"。文件名和rom名相同；
　　*如果你第一次运行rom，分析数据(SGN file) 会存在 "GBA_SIGN"；
　　*如果文件的后缀是bin，那么本软件就不会补丁，而且sav文件大小将会设置为64kb（注意，需要联动的朋友请注意）；
　　*在文件选择窗口，最高的 "SAV"将会被存为".SAV"而<1> 到 <5> 会被存为".SV1"到 ".SV5"；
　　*你最好自己建立"GBA_SAVE" 和"/BA_SIGN"
