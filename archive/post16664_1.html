最近的PSP移植，都是在画面上进行的，也大致做得差不多了，总算到了音频解码部分了。现在avg引擎用得最多的音频格式就是 OGG，因为它是免费的。我大致查了下，如果要解码OGG，需要用到libogg和libvorbis这两个库，libvorbis依赖于libogg。下面以libvorbis来说明这两个库的安装。

1. svn checkout svn://svn.ps2dev.org/psp/trunk/libvorbis 获得源代码。

2.用终端，进入下载好的源代码文件夹输入 LDFLAGS="-L$(psp-config --pspsdk-path)/lib -lc -lpspuser" 回车

3. 再输入./autogen.sh
--host psp --prefix=$(psp-config --psp-prefix) 回车

4. sudo make

5. sudo make install

就可以安装完毕了。
